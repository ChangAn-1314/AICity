# 一、项目概述

## 1.1 项目简介

**智舆**是一款面向城市治理的AI智慧舆情态势监测感知与决策推演系统，旨在通过人工智能技术赋能传统舆情监测，构建集**数据采集、智能分析、3D可视化展示、走向预测、决策模拟**于一体的综合性平台。

系统核心价值在于：**让城市管理者"看得见"舆情态势、"听得懂"民意诉求、"想得清"发展走向、"做得好"科学决策**。

本项目采用**讯飞星火4.0大模型**作为核心AI分析引擎，结合**高德地图3D可视化**、**多Agent决策推演**、**语音交互播报**等创新技术，实现从舆情监测到辅助决策的全流程智能化。系统支持**全国-省份-地级-县级**四级区域穿透分析，可根据文字、图片、视频等多模态舆情内容AI生成现场3D模型，并通过决策模拟功能帮助用户预判不同应对策略的效果。

项目首期聚焦**信阳市**场景落地，逐步扩展至河南省乃至全国城市，为政府、企业、个人提供差异化的舆情智慧服务。

## 1.2 项目背景

### 1.2.1 政策背景

**国家层面**高度重视智慧城市与数字政府建设。2025年10月，国家发改委等五部门联合印发《深化智慧城市发展推进全域数字化转型行动计划》，明确提出到2027年底建成50个以上全域数字化转型城市，深化"一网统管"建设，构建城市运行体征指标体系。《国务院关于加强数字政府建设的指导意见》确立了到2025年政府数字化履职能力框架基本形成的目标。

**河南省层面**，《河南省深化智慧城市发展推进城市全域数字化转型实施方案(2025-2027年)》提出打造3-5个国内一流的综合型城市全域数字化转型标杆。《河南省支持人工智能产业生态发展若干政策措施》提供真金白银支持：对通过国家生成式AI模型备案的企业给予100万元资金支持，设立30亿元人工智能产业基金，每年发放总规模不超过5000万元的算力券。

**信阳市层面**，《信阳市数字政府建设实施方案(2023-2025年)》确立了数字化转型的总体框架，12345热线系统已升级为舆情监测重要平台，打通省市县乡四级网络联动平台。

### 1.2.2 行业背景

**舆情监测市场高速增长**。根据行业调研数据，2024年全球舆情监测系统市场规模约23.15亿美元，预计2031年将达40.50亿美元，年复合增长率8.1%。中国市场增速领跑全球，2023年市场规模达4.2亿美元，占全球19.59%，2025年预计突破72.4亿元人民币，增速高达26.4%。

**行业痛点亟待解决**。传统舆情监测系统存在以下问题：
- **数据覆盖不全**：中小网站覆盖不足，视频/图片数据漏检率高
- **智能化程度低**：人工干预比例高，分析效率低下
- **可视化单一**：传统2D图表难以直观展示复杂舆情态势
- **决策支持弱**：只提供监测结果，缺乏走向预测和决策建议
- **下沉市场空白**：地级市/县域市场缺乏适配产品

### 1.2.3 技术背景

**大模型技术突破**为智能舆情分析提供了可能。2025年，GPT-5、讯飞星火4.0等主流大模型在情感分析基准测试上的准确率已达94.7%，远超传统方法的75-85%。多模态分析能力显著提升，支持文本、图片、音频、视频等多种媒体形式的综合分析。

**3D可视化技术成熟**。WebGL、Three.js、高德地图JS API 2.0等技术已广泛应用于三维城市可视化场景，支持3D建筑高亮、热力图叠加、区域边界渲染等功能。

**AI 3D生成技术兴起**。Tripo AI、TripoSR等工具可在10秒内将文字/图片转换为3D模型，为舆情场景还原提供技术支撑。

### 1.2.4 地域特色

**信阳市**作为河南省地级市，具有独特的区位优势和发展需求：
- **鄂豫皖革命老区**：红色文化资源丰富，舆情管理需兼顾文化传承
- **茶乡文化**：信阳毛尖品牌舆情关注度高，涉农舆情监测需求明确
- **人口大市**：常住人口超600万，民生诉求多元，舆情治理任务重
- **数字化转型起步**：12345热线平台已建成，但智能化水平待提升

本项目立足信阳本土需求，充分发挥地域特色优势，探索"AI+城市"智慧治理新路径。

### 2.1.2 功能模块

系统包含**六大核心功能模块**：

**模块一：实时舆情监测**
- 多源数据采集：覆盖微博、抖音、小红书、B站、微信公众号等主流平台
- 多模态解析：支持文字、图片、视频内容的智能识别与分析
- 实时预警推送：分钟级舆情发现，多渠道预警通知

**模块二：智能舆情分析**
- 情感极性分析：正面/负面/中性情感识别，准确率超90%
- 主题聚类：自动识别舆情热点话题，生成话题云图
- 传播路径追踪：分析舆情传播链路，识别关键传播节点
- 影响力评估：综合评估舆情热度、传播范围、影响程度

**模块三：3D地图可视化**
- 四级区域穿透：全国→省份→地级市→县级，逐级下钻
- 3D建筑高亮：舆情热点区域建筑物高亮标注
- 热力图叠加：舆情分布密度可视化
- 飞行动画：长距离城市切换平滑过渡

**模块四：AI场景还原**
- 文字转3D：根据舆情文字描述生成现场3D模型
- 图片转3D：将舆情图片重建为三维场景
- 地图叠加：将生成的3D模型放置在对应地理位置

**模块五：走向预测与决策模拟**
- 趋势预测：基于历史数据和AI分析预测舆情走向
- 多场景推演：乐观/中性/悲观三种场景预测
- 决策建议：生成多套应对方案供用户选择
- 效果模拟：模拟不同决策的执行效果

**模块六：语音交互与播报**
- 语音预警：重要舆情语音播报提醒
- 语音指令：支持"展示信阳市舆情热点"等语音操作
- 智能问答：语音咨询舆情相关问题

## 2.2 项目特色

### 2.2.1 核心创新点

**创新点一：AI驱动的舆情场景3D还原**

这是本项目最具创新性的功能。传统舆情系统只能展示文字、图表等抽象信息，用户难以直观感知事件现场。智舆系统创新性地引入**AI 3D生成技术**，可根据舆情内容自动生成现场3D模型：

- **文字舆情**：通过讯飞星火提取场景描述，调用Tripo AI生成3D模型
- **图片舆情**：利用TripoSR/Tripo单图重建技术生成3D场景
- **视频舆情**：提取关键帧进行3D重建，结合语音转写生成完整场景

生成的3D模型可直接叠加到高德地图对应位置，实现**"在地图上看见新闻现场"**的沉浸式体验。

```
舆情内容输入
    │
    ├─→ [文字舆情] ──→ 讯飞星火提取场景 ──→ Tripo AI ──→ GLB模型
    │
    ├─→ [图片舆情] ──→ TripoSR ──→ 3D模型
    │
    ├─→ [视频舆情] ──→ 关键帧提取 + 语音转写 ──→ 3D场景
    │
    └─→ [输出] ──→ Three.js渲染 + 高德地图叠加 ──→ 沉浸式展示
```

> **[图8：AI 3D场景还原技术流程图]** 建议制作流程图展示从输入到输出的完整技术路径。

**创新点二：多Agent协作的决策模拟系统**

基于**LangGraph**框架构建多Agent协作体系，实现从舆情分析到决策模拟的全链路智能化：

| Agent | 职责 | 输入 | 输出 |
|-------|------|------|------|
| **分析Agent** | 舆情深度分析 | 原始舆情数据 | 分析结果(情感、主题、影响力) |
| **预测Agent** | 走向趋势预测 | 分析结果+历史数据 | 多场景预测(乐观/中性/悲观) |
| **决策Agent** | 生成应对方案 | 预测结果 | 多套决策建议 |
| **模拟Agent** | 模拟执行效果 | 用户选择的决策 | 效果评估报告 |

用户可以：
1. 查看AI生成的多套决策方案
2. 选择或自定义决策内容
3. 启动模拟，观看决策执行效果的动态推演
4. 对比不同决策的优劣，辅助最终决策

> **[图9：多Agent决策模拟架构图]** 建议制作四Agent协作流程图，展示数据流转关系。

**创新点三：四级区域穿透的3D地图可视化**

突破传统舆情系统的2D展示局限，实现**全国→省份→地级市→县级**的四级3D可视化穿透：

- **全国视图(Zoom 4-6)**：展示全国舆情热度分布，国境线霓虹描边
- **省份视图(Zoom 6-8)**：展示省内各地市舆情对比，省界高亮
- **地市视图(Zoom 8-12)**：展示城市内部舆情分布，关键建筑3D标注
- **县区视图(Zoom 12+)**：展示街道级舆情详情，支持点击查看原文

配合**高空跳跃动画**，用户可在不同层级间平滑过渡，获得沉浸式的数据探索体验。

# 三、项目可行性

## 3.1 技术可行性

### 3.1.1 核心技术说明

**大语言模型技术**

讯飞星火4.0 Ultra作为本项目核心AI引擎，具备以下能力：
- **超长上下文**：128K tokens，可处理长篇舆情分析报告
- **中文优化**：针对中文语境深度优化，网络用语理解能力强
- **多任务适配**：支持情感分析、主题提取、摘要生成、对话问答等多种任务
- **API成熟稳定**：提供完善的REST API和WebSocket接口

国产大模型在舆情分析领域已达到实用化水平。根据行业评测，2025年主流系统情感识别准确率普遍超过90%，对"破防了""躺平"等新兴网络用语的情感极性判断准确率超92%。

**3D可视化技术**

- **高德地图JS API 2.0**：原生支持3D地图视图、建筑层渲染、区域边界查询
- **Three.js**：成熟的WebGL框架，与高德地图GLCustomLayer无缝对接
- **WebGL**：浏览器原生支持，无需插件，兼容性好

技术选型已在前端原型中得到验证，地图渲染流畅、动画效果平滑。

**AI 3D生成技术**

- **Tripo AI**：支持文字/图片/草图转3D，生成速度约10秒，输出PBR材质模型
- **TripoSR**：Stability AI开源模型，单图转3D，PyTorch实现，可本地部署
- **3D Gaussian Splatting**：实时场景重建技术，适用于视频转3D场景

**小模型微调技术**

- **LoRA**：将微调参数量降低90%以上，消费级GPU即可训练
- **P-Tuning**：通过提示向量引导模型，适合多任务共享
- **QLoRA**：4-bit量化+LoRA，进一步降低存储需求

这些技术使大学生团队可以用有限资源实现专属领域的模型优化。
