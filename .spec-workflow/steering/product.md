# Product Overview

## Product Purpose

**智舆** 是一款 AI 城市舆情态势监测感知与决策推演系统，旨在解决城市管理者在舆情监测、分析和决策方面面临的核心挑战：

- 舆情信息分散在多个平台，难以统一监测
- 传统舆情分析效率低，无法实时响应
- 缺乏直观的可视化手段呈现舆情态势
- 决策缺乏 AI 辅助，难以预测舆情走向
- 舆情现场难以直观还原，缺乏沉浸式感知

本系统通过 AI 大模型 + 3D 可视化 + 多 Agent 决策模拟的技术组合，为用户提供从舆情监测、智能分析、趋势预测到决策模拟的全链路解决方案。

## Target Users

### 主要用户群体

1. **政府部门**

   - 需求：城市舆情监测、突发事件预警、决策支撑
   - 痛点：舆情响应慢、信息不对称、决策缺乏数据支撑

2. **企业机构**

   - 需求：品牌舆情监测、危机公关预警、市场情绪分析
   - 痛点：舆情发现滞后、分析能力不足、决策反应慢

3. **个人用户**
   - 需求：本地热点关注、区域舆情了解
   - 痛点：信息碎片化、缺乏聚合分析工具

### 应用场景层级

- **单城市**：以信阳市为示范城市实现完整功能
- **省级**：扩展至河南省所有城市
- **全国**：未来展望覆盖全国范围

## Key Features

1. **实时舆情监测**

   - 多平台数据采集：社交媒体、新闻媒体、自媒体等
   - 多形式内容支持：文字、图片、视频
   - 实时推送与预警机制

2. **AI 智能分析**

   - 基于讯飞星火 4.0 的舆情分析
   - 情感分析与关键词提取
   - 微调小模型专注细分领域

3. **3D 地图可视化**

   - 高德地图 3D 展示舆情态势
   - 区域热力图与建筑高亮
   - 全国 -> 省份 -> 地级 -> 县级层级穿透

4. **AI 现场还原**

   - 根据文字/图片/视频生成 3D 场景模型
   - 舆情事件现场沉浸式呈现
   - 模型与地图叠加展示

5. **走向预测与决策模拟**

   - AI 预测舆情发展趋势
   - 多场景决策方案生成
   - 用户可自定义决策进行模拟推演

6. **语音交互**
   - 讯飞 TTS 语音合成播报预警
   - 讯飞 ASR 语音输入交互
   - 实时语音流式对话

## Business Objectives

- **比赛目标**：在讯飞杯比赛中取得优异成绩，充分展示讯飞技术集成能力
- **技术验证**：验证 AI + 3D 可视化 + 多 Agent 决策在舆情分析领域的可行性
- **原型打磨**：构建可演示、可扩展的产品原型
- **能力展示**：展示大学生团队的技术创新和工程能力

## Success Metrics

- **功能完整度**：核心功能模块全部实现并可演示
- **讯飞技术集成**：至少集成 3 项讯飞技术（星火大模型、TTS、ASR）
- **可视化效果**：3D 地图流畅运行，视觉效果达到展示标准
- **用户体验**：交互流程完整，响应时间 < 3 秒
- **代码质量**：结构清晰，可维护，可扩展

## Product Principles

1. **讯飞优先**：优先集成讯飞技术（星火、TTS、ASR），作为比赛加分项
2. **可视化驱动**：以 3D 可视化为核心呈现手段，追求直观震撼的展示效果
3. **AI 赋能**：充分利用大模型能力提升分析和决策智能化水平
4. **渐进实现**：从单城市到省级到全国，分阶段实现功能扩展
5. **大学生可行**：技术选型考虑学习成本和资源限制

## Monitoring & Visibility

- **Dashboard Type**: Web-based，响应式设计支持多端访问
- **Real-time Updates**: WebSocket 实时推送舆情更新
- **Key Metrics Displayed**:
  - 舆情热度指数
  - 情感倾向分布
  - 区域舆情地图
  - 趋势预测曲线
  - 预警通知列表
- **Sharing Capabilities**: 舆情报告导出、截图分享

## Backend Services

### API 服务

后端采用 FastAPI 构建 RESTful API，提供以下核心服务：

1. **舆情数据服务**
   - 热点舆情 CRUD
   - 舆情事件时间线
   - 地域统计数据

2. **AI 分析服务**
   - 讯飞星火舆情分析
   - 情感倾向识别
   - 关键词提取

3. **决策模拟服务**
   - LangGraph 多 Agent 编排
   - 趋势预测引擎
   - 决策方案生成

4. **3D 场景服务**
   - Tripo AI 文字/图片转 3D
   - 模型状态查询
   - GLB 文件存储

5. **数据采集服务**
   - 采集任务管理
   - 多源数据聚合
   - 定时采集调度

6. **语音交互服务**
   - 讯飞 TTS 语音播报
   - 讯飞 ASR 语音输入
   - 流式语音处理

### 实时通信

- **WebSocket 推送**: 舆情更新、预警通知实时下发
- **事件类型**: 新舆情、热度变化、预警触发、分析完成

### 异步任务

- **Celery 任务队列**: 舆情采集、AI 分析、模型生成等耗时操作
- **定时调度**: APScheduler 定时采集和统计报表
